pwd = prints the absolute path name of the current working directory.
ls = Display the contents list of your current directory.
cd ~ = changes the working directory to the user’s home directory.
ls -l = current directory contents in a long format
ls -la = Display current directory contents, including hidden files (starting with .). Use the long format.
ls -i = Display current directory contents.
mkdir = creates a directory
mv = move and rename a file or directories
rm = delete a file or a directory
cd - = script that changes the working directory to the previous one.
ls -la = script that lists all files
